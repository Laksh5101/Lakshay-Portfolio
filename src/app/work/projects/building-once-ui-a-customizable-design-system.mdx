## Overview

Developed an interactive Resume Analyzer web application using Streamlit, enabling users to upload resumes and receive skill and keyword extraction insights.

Key Features

- **Resume Upload**: Supports multiple formats including PDF and DOCX.
- **Text Extraction**: Extracts text content from uploaded resumes using Python libraries like PyPDF2 and python-docx.
- **Keyword & Skill Extraction**: Uses natural language processing (NLP) with spaCy and custom dictionaries to identify key skills, experiences, and qualifications.
- **Visual Insights**: Displays summary statistics and keyword frequency charts for easy resume evaluation.
- **User-friendly UI**: Built with Streamlit for easy interaction without coding knowledge.

## Technologies Used

- **Python**: Core language for processing and analysis.
- **Streamlit**: Framework for building the interactive web interface.
- **NLP**: spaCy and Regex for text parsing and keyword extraction.
- **File Handling**: PyPDF2 and python-docx for resume text extraction.

## Challenges and Learnings

- Learned how to handle various file formats and extract text consistently.
- Implemented efficient keyword matching and text processing for resume analysis.
- Improved skills in building simple, interactive web apps with Streamlit.

## Outcome

- Delivered a functional web app that provides immediate, actionable feedback on resumes.
- Useful tool for job seekers to optimize resumes and for recruiters to screen candidates quickly.
